**K 번째 수 찾기**

먼저 프로그래머스 문제였기 떄문에,

파라미터, 코드 내에서 사용할 변수의 이름이 정해져 있어서 그대로 작성하였습니다.

해당 문제는 

첫, 마지막 인덱스, 그리고 찾는 수의 인덱스로 

원하는 수를 찾기 위해서 3개의 변수가 필요하고, 

해당 변수들이 총 3개의 배열의 인자로 주어지게 됩니다.

또한 이를 위해서 command 배열이 2차원 배열로 되어있음을 확인하였습니다.

먼저, 배열에서 원하는 수를 순서대로 찾기 위해서 반복문을 

command의 길이로 설정하였고,

해당 3개의 변수를 command 배열로부터 받아오는 코드를 작성하였습니다.

변수를 가져온 후에는

계산과정에 필요한 새로운 배열인 NewArray를 선언하고, 

copyOfRange 를 사용하여 첫 인덱스, 마지막 인덱스로 

배열을 자르고, 

또한 자른 배열을 sort를 사용하여 정렬하는 과정을 거쳤습니다.

배열을 원하는대로 자르고 정렬하지 않는다면, 문제 설명에 나온대로 

원하는 인덱스의 위치에 있는 수를 가져올 수 없기 때문에 sort를 통해서

배열을 정렬하고, 배열을 자른 결과물에서 k-1 번째의 수를 answer 배열에

저장하고, 반복문을 벗어나서 답을 반환하는 코드를 작성하였습니다.

그렇게 작성한 후, 프로그래머스 사이트 내에서 실행해보니

결과는 성공이었습니다.
