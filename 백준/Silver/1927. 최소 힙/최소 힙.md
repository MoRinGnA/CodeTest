해당 문제는 교수님께서 말씀하신 것처럼 아주 기본적인

최소 힙을 그냥 구현해보는 문제로서,

배열에 자연수를 입력받고

해당 자연수의 크기만큼 반복문을 수행하게 작성합니다.

또한 사용자의 입력값을 변수 a에 순서대로 저장한 후

해당 값이 0이고, 큐가 비어있다면, StringBuilder에 0과 \n을

append를 통해 삽입하고,

큐가 비어있지 않다면 poll 메소드를 통해서 최솟값을

큐에서 삭제함과 동시에 SB에 저장합니다.

위 조건에 부합하지 않고, a에 저장된 값이 자연수라면

add를 통해서 큐에 변수 내부에 저장된 값을 삽입하게 됩니다.

위 과정에서 큐 내부는 자동으로 정렬이 진행되고,

반복문이 종료된다면 SB에 저장된 값을 차례대로 출력하면서

코드가 마무리되는 방식으로 작성하였습니다.

그 결과 정답이었습니다.
